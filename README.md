# 🎮 AVOID LASER : DODGE GAME

> C언어로 구현한 콘솔 기반 레이저 회피 게임 (2학년 1학기 텀프로젝트 | 2025.06)

---

## 📖 프로젝트 소개

사방에서 날아오는 레이저를 피해 최대한 오래 생존하는 콘솔 게임입니다.
더블 버퍼링을 직접 구현하여 화면 깜빡임 없이 부드러운 렌더링을 구현했습니다.

---

## 🕹️ 게임 방법

- **방향키**로 플레이어(`♣`) 이동
- 중심부 사각형 테두리를 따라 이동 가능
- 4방향에서 날아오는 레이저를 피해 생존
- 방향키를 누른 순간부터 플레이 타임 측정

---

## ⚙️ 주요 기능

| 기능 | 설명 |
|------|------|
| 더블 버퍼링 | 화면 깜빡임 없는 부드러운 렌더링 구현 |
| 난이도 상승 | 10초마다 레이저 속도 자동 증가 |
| heart 아이템 (`♥`) | 획득 시 레이저에 한 번 맞아도 생존 |
| slow 아이템 (`∞`) | 획득 시 3초간 레이저 속도 감소 |
| 아이템-레이저 충돌 | 레이저와 아이템이 충돌하면 아이템 소멸 |
| 엔딩 화면 | SCORE(플레이 타임) 확인, 별 랜덤 출력 |
